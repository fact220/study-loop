# Changelog

All notable changes to Study Loop will be documented in this file.

## [1.0.0] - 2026-05-04

### 🎉 Initial Release

#### Core Features
- **AB Loop Playback** — Set A/B points on any YouTube video and loop automatically
- **Fine-tuning** — Adjust A/B points by 0.5 seconds (Shift+click for 0.1s precision)
- **Segment Saving** — Save loop sections with custom names
- **Tagging** — Tag segments as Weak / Important / Needs Review / None
- **Mastery Tracking** — Track segments as Unlearned → Learning → Mastered

#### Organization
- **Course Management** — Create courses for different subjects (e.g., "Real Estate Exam")
- **Folder System** — Organize videos into folders within courses
- **Video Cards** — View segment count, mastery status, and tags per video
- **Cross-course Video Moving** — Move videos between courses and folders

#### Study Tracking
- **Monthly Calendar** — Heatmap showing daily study time with Japanese date format
- **Study Statistics** — Total study time, mastery rate, consecutive study days
- **Exam Countdown** — Days remaining until your exam date
- **14 Achievement Badges** — Motivational milestones for study consistency

#### User Experience
- **3 Color Themes** — Dark, Light, Natural
- **Loop Interval** — Pause between loops (0–10 seconds) for active recall
- **Keyboard Shortcuts** — A, B, L, N, P for fast control
- **Side Panel** — Full learning management alongside YouTube
- **Popup Dashboard** — Quick stats and recent segments

#### Data & Privacy
- **Local Storage Only** — All data stored in chrome.storage.local
- **Data Export/Import** — Backup and restore as JSON
- **Multilingual** — English and Japanese (auto-detected)
- **No Account Required** — Start using immediately

#### Monetization
- **Free Plan** — 5 saved segments, all features included
- **Pro Plan** — Unlimited segments (¥300/month or ¥2,400/year)
